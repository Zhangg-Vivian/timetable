# timetable
##项目介绍
制作了一个课程表，通过添加课程来完善主页面的课程表，并添有网络请求功能
##核心功能
- 本地课程管理：手动添加课程（名称、时间、地点、老师），通过SQLite数据库持久化保存；
- 课程展示：5列网格布局展示所有课程，文字自动换行，适配窄列显示；（由于排版过窄，把周末两天砍掉了）
- 页面跳转：主页点击按钮跳转到添加课程页面
